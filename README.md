# Android Penetration Testing Lab
## Overview
This repository contains the documentation and methodology for a penetration testing lab
conducted on an Android 8.0 environment. The project focuses on understanding
vulnerabilities related to unauthorized remote access and post-exploitation techniques using
the Metasploit Framework.

## Contents
- Report.pdf: Complete lab report with execution steps and screenshots.

## Tools Used
- Kali Linux
- Metasploit Framework
- Android 8.0 (Emulator)

## Methodology
The testing involved:
1. Payload generation using msfvenom.
2. Setting up a multi-handler for connection.
3. Analyzing session stability and post-exploitation commands.

## Disclaimer
This project is strictly for educational and research purposes only. All testing was performed in
a controlled, isolated environment.
